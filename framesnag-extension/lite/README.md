# FrameSnag Lite — ScreenMerch integration

Unpacked Chrome extension build for ScreenMerch creators. Load this folder in Chrome → Extensions → Developer mode → Load unpacked.

## ScreenMerch features

- `screenmerch-sync.js` — reads dashboard **Save to** list from `screenmerch.com` / subdomains and stores `list_id` for FrameSnag.
- `popup-lite.js` — opens Favorites with `?tab=favorites&list_id=…` so paste/upload targets umbrella collaborator pages.

Visit **Dashboard → Favorites** and set **Save to** before snagging on YouTube.

See also `docs/FRAMESNAG_INTEGRATION_URL.md` in the repo root.
